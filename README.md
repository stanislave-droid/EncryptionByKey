# EncryptionByKey

<h1>UA</h1>
<p>Програма для шифрування, напочатку вводиш ім'я файлу, в якому зберігається данні для шифрування/дешифрування з типом .data </p>
<p>Вводиться ключ для шифрування, у якого довжина не перебільшує 100 символів</p>
<p>Вибираєш метод шифрування, мультипотоки чи однопотік (малі об'єми даних однопотік шифрує/душифрує швидше)</p>
<p>Дані шифруються та зберігаються у новому файлі, у якого введена назва + "_enc" або "_dec", якщо дешифрування, + ".data" </p>

<h1>EN</h1>
<p>Encryption program, first you enter the name of the file in which the data for encryption/decryption is stored with the type .data </p>
<p>Enter the encryption key, the length of which does not exceed 100 characters</p>
<p>Choose the encryption method, multi-threaded or single-threaded (single-threaded encrypts/decrypts small amounts of data faster)</p>
<p>The data is encrypted and saved in a new file, the name of which is entered + "_enc" or "_dec", if decryption, + ".data" </p>
