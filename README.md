# EncryptionByKey

<p>Програма для шифрування, напочатку вводиш ім'я файлу, в яком зберігається данні для шифрування/дешифрування з типом .data </p>
<p>Вводиться ключ для шифрування, у якого довжина не перебільшує 100 символів</p>
<p>Вибираєш метод шифрування, мультипотоки чи однопотік (малі об'єми даних однопотік шифрує/душифрує швидше)</p>
<p>Дані шифруються та збурігаються у новому файлі, у якого введена назва + "_enc" + ".data" </p>
